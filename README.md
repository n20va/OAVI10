### Исходные данные

- Синтезируемая фраза: `Хорошо живёт на свете Винни-Пух`.

### Теоретическая основа
Фонема рассматривается как минимальная звуковая единица, а аллофон — как вариант её реализации. Для гласных используется тональный источник и формантные фильтры, для согласных — шумовой или смешанный источник.

### Формулы
```text
X(m,k) = Σ x[n] w[n-mR] exp(-j 2πkn/N)
P(m,k) = |X(m,k)|²
y[n] = Σ BPF_i(source[n], F_i, Q_i)
crossfade = left·(1-a) + right·a, a ∈ [0,1]
```
